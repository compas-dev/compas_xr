    Storage ¤

    Storage(config_path: str)

    A Storage is defined by a Firebase configuration path and a shared storage reference.

    The Storage class is responsible for initializing and managing the connection to a Firebase Storage. It ensures that the storage connection is established only once and shared across all instances of the class.

    Parameters:

    • config_path (str) –

      The path to the Firebase configuration JSON file.

    Attributes:

    • config_path (str) –

      The path to the Firebase configuration JSON file.
